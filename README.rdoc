== README

Please Follow the Steps to run the Application :

- Clone the Repository to your Local Machine

- Got to the path to your repository on you Local Machine

- Run the following command to install Gems : <tt>bundle install</tt>

- To create the database schema : <tt>rake db:create</tt>
  
- To alter the database schema : <tt>rake db:migrate</tt>
  
- To add data to the database schema : <tt>rake db:seed</tt>
  
- To run the server : <tt>rails server / rails s</tt>
